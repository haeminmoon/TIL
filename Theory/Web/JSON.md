#JSON

##JSON이란?

서로 다른 언어들간에 데이터를 주고 받는 여러 방법이 있다. 대표적인 것이 XML인데, XML은 문법이 복잡하고, 
엄격한 표현규칙으로 인해서 json 대비 데이터의 용량이 커진다는 단점이 있다.

JSON은 경량의 데이터 교환 형식으로 JavaScript에서 숫자와 배열등을 만드는 형식을 차용해서 
이것을 다른 언어에서도 사용할 수 있도록 한 텍스트 형식이다.
